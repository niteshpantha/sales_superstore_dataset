Sales Superstore Analytics — SQL & Python EDA

A complete SQL and Python EDA project analyzing sales data from a sample Superstore dataset. This project demonstrates data cleaning, database design, analytics, and business insights from beginner to advanced levels.


Project Overview
This project covers:

SQL: Normalized database, complex queries, CTEs, joins, aggregates, and window functions.
Python EDA: Data exploration, cleaning, aggregation, and visualization using pandas, numpy, and matplotlib.
Business Insights: Sales trends, top customers, product performance, customer segmentation, shipping analysis, and regional contributions.


Key Business Questions Solved
﻿Beginner Level
    1. What is the total sales in the dataset?
    2. How much did each category sell in total?
    3. Which customers generated the highest sales?
    4. How many orders are there for each customer segment?
    5. What is the total sales in the South region?
Intermediate Level
    1. What are the monthly sales trends?
    2. Which products had the highest sales?
    3. How many orders were shipped via each shipping mode?
    4. What is the average sales per customer?
    5. Which customers placed the most orders?
Advanced Level
    1. 1. Calculate the total sales per category per region.
    2. Identify repeat customers and compare their total sales to single-time customers.
    3. Calculate days between order_date and ship_date and find the fastest shipping mode.
    4. What percentage of total sales comes from each region?
    5. Compare year-over-year sales growth and identify the year with the highest growth.
    6. Identify customers who buy from more than one category.

    
Skills Demonstrated

SQL: Joins, CTEs, aggregates, window functions, business-focused queries
Python / EDA: Data cleaning, exploration, correlation analysis, visualization
Data Modeling: Normalized PostgreSQL database with customers, orders, products, categories, and shipping
Business Analysis: Translating raw data into actionable insights

This project showcases practical analytics skills, from database design to actionable business insights, making it ideal for a data analyst portfolio.
